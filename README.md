Smart Calculator (Python)

A simple menu-driven calculator built using Python.

Features:

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Continuous execution until exit

Concepts Used:

- while loop
- if-elif-else conditions
- user input handling

How to Run:

1. Run the Python file
2. Choose an operation from the menu
3. Enter numbers
4. Get result
